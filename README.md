A DIY smart glasses project powered by an ESP32 and a transparent OLED display.
Displays real-time information like weather, WiFi status, and system data directly in your field of view.

 -Features
 -WiFi connectivity (ESP32)
 -Live weather data (API-based)
 -Transparent OLED display output
 -Lightweight + wearable design
 - Fully customizable code
 
  Hardware that will be used
-ESP32 development board
-Transparent OLED display (SPI)
-Wires / breadboard or custom PCB
-Power source (battery pack or LiPo)
-Frame (cardboard or 3-d printed)
-Raspberry pi 0 2w
-Pressure sensor
-Micro Phone module
-Speaker
